Jualo give user easy access to their Jualo.com account.

== Description ==

Jualo give easy access to your Jualo Accounts so you can check your profile, or your posted ads. 

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Jualo screen to configure the plugin
4. Enable the Display Jualo checkbox, and enter your Jualo Frame that can you get from https://www.jualo.com/profile/widgets
